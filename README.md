This project is a simple 2D space shooter game built using the Godot Engine. The game focuses on basic gameplay mechanics such as player movement, shooting, and enemy interaction. Even though it is a beginner project, it demonstrates fundamental game development concepts including scene management, animation, and input handling. This project was created as a learning step to understand how a game is structured and how different systems work together.

You can download or explore the project directly through this GitHub repository: https://github.com/Ryan6712/Space-shooter/tree/main. Feel free to clone the repository and run it using Godot to see how it works, or modify it as you like for learning purposes. This project is open for improvement and serves as a foundation for more complex games in the future.
